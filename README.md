# Expense Tracker

A Python-based CLI expense tracker for monitoring and analyzing personal spending habits.

## Features 
- Tracks daily expenses
- Categorizes expenses into:
  - Food
  - Transpo
  - Bills
  - Discretionary
  - Others
- Search expenses by date or description
- Generate spending summaries and expenditure analysis

  ## Installation

  ```bash
  git clone https://github.com/fonsi31/expense-tracker.git
  ```

  ## Technologies Used

  - Python
  - JSON
  - Matplotlib

## How to Run

```bash
venv\Scripts\activate
python expense_tracker.py
```
